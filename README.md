## Projects 🏆

**Bachelor's Thesis — Vaalit (2026)**, a client-side visualization tool for Finnish election data (municipal, parliamentary, presidential & European Parliament) built with Leaflet + Chart.js and four original shared libraries, following a Design Science Research methodology. [Link to repo](https://github.com/JusefA/Finnish-Election-Visualize)

**Is It Halal**, a free non-profit web app for classifying halal dietary information. It is now home to a growing community of active users. [Link to website](https://isithalal.vercel.app/)

**Finance Copilot**, a SaaS finance dashboard for small Finnish construction & trades companies — Next.js + Supabase + Vercel + Paddle (EU VAT) + OpenAI, with magic-link auth. [Link to website](https://finance-copilot-web3.vercel.app/)

**Cloud Drive**, a full-stack document management app with a Node.js / Express / TypeScript backend, React / Vite frontend, MongoDB, and JWT auth.[Link to repo](#)
